Create an Angular application that:
  • Fetches data from GitHub's GraphQL API.
  • Displays a tabular view of the data using AG Grid.
  • Visualizes a specific aspect of the data using D3.js.
Requirements:
  • GitHub API Integration: Use GitHub's GraphQL API to fetch a list of repositories for a specific user.
  • AG Grid: Display repository data in a tabular format with features like sorting, filtering, and pagination.
  • D3.js: Create a bar chart to visualize the number of stars each repository has.
Instructions:
  • Setup the Project:
    • Use Angular CLI to generate a new Angular project.
    • Install necessary packages: angular/core, angular/common, angular/http, ag-grid-angular, d3,ngrx, apollo-angular, and any other packages deemed necessary.
  • GitHub GraphQL API:
    • Register an OAuth application on GitHub to obtain a personal access token.
    • Use Apollo Angular to interact with GitHub's GraphQL API.
  • AG Grid Integration:
    • Create an Angular component to display the data in a grid.
  • Search Input: Implement a search input field that allows users to filter the repositories displayed in the AG Grid.
  • State Management with NgRx: Use NgRx to manage the state of the application, including fetching repositories, storing them in the state, and filtering the data based on user input. Use NgRx SignalStore for bonus points
  • D3.js Visualization:
    • Create an Angular component for the bar chart visualization.
  • Combine Everything:
    • Fetch data using a service and pass it to both AG Grid and D3 components.
    • Ensure a cohesive and responsive design.
Submission:
  • The project should be hosted in a public GitHub repository.
  • Provide a README file with setup instructions.
  • Ensure the application is well-documented, well structured and code is clean.
  • Consider creating separate, routed components to display the table and the data viz.
